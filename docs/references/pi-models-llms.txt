# Pi Available Models

This file lists all built-in models available in pi (as of 2025-06-05), grouped by provider.
Use `pi --list-models` to get the current live list.

Use `/model` (or Ctrl+L) in interactive mode, or `--model <pattern>` on the CLI to select a model.

## Model Selection Tips

- **Best coding overall**: claude-sonnet-4.6, claude-opus-4.8, gpt-5.5, gpt-5.4
- **Fastest**: claude-haiku-4.5, grok-code-fast-1, gemini-3.5-flash
- **Largest context (1M)**: deepseek-v4-flash/pro, claude-opus-4.6, claude-sonnet-4.6, mimo-v2.5, qwen3.7-max
- **Image understanding**: All Claude, Gemini, and GPT models; kimi-k2.5/6, mimo-v2.5, qwen3.6-plus
- **Thinking (extended reasoning)**: All listed models except gemini-2.5-pro, gpt-4.1, gpt-4o
  - Use `--thinking <level>` (off/minimal/low/medium/high/xhigh) or Shift+Tab in interactive mode

## Adding Custom Models

Add your own models via `~/.pi/agent/models.json`. See pi docs for details:
- Provider setup: https://docs.pi.dev/providers (or `pi docs/providers.md`)
- Custom models: `pi docs/models.md`
