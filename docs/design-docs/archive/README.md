# ADR Archive

This directory holds ADRs that are no longer part of the current decision surface.

Archive an ADR when one of these is true:

- the decision was replaced by a newer architecture
- the document mainly records migration process history rather than a still-relevant decision
- keeping it in the main ADR index would mislead future development

Archived ADRs are still useful for historical context and rollback analysis, but should not be treated as the default guidance for new work.
